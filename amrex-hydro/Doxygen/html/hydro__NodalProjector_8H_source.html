12//
13//
14// *************************** DEFAULT MODE ***************************
15//
16// Solves
17//
18// div(sigma*grad(phi)) = div(vel) + S_nd + S_cc
19//
20// and performs the projection
21//
22// vel = vel - sigma * grad(phi)
23//
24// vel, sigma, and S_cc are cell-centered variables, while
25// phi and S_nd are nodal-centered variables.
26//
27// *************************** CUSTOM MODE ***************************
28//
29// Solves
30//
31// div(sigma*grad(phi)) = rhs
32//
33// and performs the projection
34//
35// vel = vel - (sigma/alpha) * grad(phi)
36//
37// alpha is a cell-centered variable, while rhs is nodal-centered.
38//
39// In this mode, the user provides rhs and alpha
40//
41// By default alpha is assumed to be 1. Use setAlpha to change the default.
42//
43// Use setCustomRHS to provide a custom RHS, else "div(vel) + S_nd + S_cc"
44// is used.
45//
46// Example: rhs = div(alpha*vel)
47//
